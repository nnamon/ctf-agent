CTF SANDBOX TOOLS REFERENCE
============================
All tools pre-installed in the sandbox. cat /tools.txt to view this file.

== NETWORKING ==
nc (netcat)        TCP/UDP connections. nc host port <<'EOF' for heredoc sessions.
curl               HTTP client. -s silent, -k skip TLS, -d data, -H header.
wget               HTTP download. wget -O file url.
nmap               Port scanner. nmap -sV host, nmap -p- host.

== WEB PENTESTING ==
ffuf               Fast web fuzzer. ffuf -u http://t/FUZZ -w /opt/wordlists/common.txt
                   Param fuzzing: ffuf -u 'http://t/?FUZZ=x' -w wordlist
                   VHost: ffuf -u http://t/ -H 'Host: FUZZ.t' -w wordlist -fs 0
sqlmap             Automated SQL injection. sqlmap -u 'http://t/?id=1' --batch --dbs
                   POST: sqlmap -u 'http://t/login' --data 'u=a&p=b' --batch
/opt/wordlists/    common.txt, raft-medium-directories.txt, raft-medium-files.txt,
                   quickhits.txt, passwords-top10k.txt (subset of SecLists)

== BINARY ANALYSIS ==
file               Identify file type. file binary
strings            Extract printable strings. strings -n 8 binary
xxd                Hex dump. xxd file | head, xxd -r to reverse.
hexdump            Hex dump. hexdump -C file.
binwalk            Firmware/embedded extraction. binwalk -e file to extract.
objdump            Disassembly. objdump -d binary (from binutils).
readelf            ELF header info. readelf -a binary.
nm                 Symbol listing. nm binary.

== JVM / JAVA ==
javac / java       OpenJDK compiler/runtime (default-jdk).
javap              Bytecode disassembler. javap -c -p ClassName, javap -v -p Class.
jar                JAR archive tool. jar tf app.jar, jar xf app.jar.
jadx               Java/APK decompiler. jadx -d out/ app.jar
                   GUI variant available as jadx-gui (X required).
keytool            Certificate/keystore tool.

== DEBUGGERS / TRACING ==
gdb                GNU debugger. gdb binary, then run, break, info, x/...
ltrace             Library call tracer. ltrace ./binary
strace             System call tracer. strace ./binary
radare2            Reverse engineering. r2 binary, then aaa; pdf @main; afl

== STEGANOGRAPHY ==
steghide           Embed/extract in JPEG/BMP/WAV. steghide extract -sf file.jpg
stegseek           Brute-force steghide passwords. stegseek file.jpg wordlist.txt
zsteg              PNG/BMP steg detector. zsteg file.png (tries LSB, bit planes, etc.)
exiftool           EXIF/metadata reader. exiftool file.jpg
pngcheck           PNG structure checker. pngcheck -v file.png

== IMAGE / MEDIA ==
identify           ImageMagick: identify file properties. identify image.png
convert            ImageMagick: transform images. convert in.png -enhance out.png
montage            ImageMagick: tile images. montage *.png out.png
tesseract          OCR. tesseract image.png stdout, or --psm 6 for block text.
ffmpeg             Audio/video processing. ffmpeg -i in.wav out.mp3
sox                Audio analysis. sox in.wav -n spectrogram -o spec.png

== FILESYSTEM FORENSICS ==
mmls               Sleuthkit: partition layout. mmls disk.img
fls                Sleuthkit: list files (incl. deleted). fls -r -o offset disk.img
icat               Sleuthkit: extract file by inode. icat -o offset disk.img inode > out
tsk_recover        Sleuthkit: recover deleted files. tsk_recover disk.img outdir/
fsstat             Sleuthkit: filesystem stats. fsstat -o offset disk.img
foremost           File carving by header/footer. foremost -i disk.img -o outdir/
testdisk           Partition recovery. testdisk disk.img
xfs_db             XFS debugger. xfs_db -c "sb 0" -c "p" disk.img
xfs_repair         XFS repair. xfs_repair -n disk.img (read-only check).
dcfldd             Forensic dd with hashing. dcfldd if=in of=out hash=md5
volatility3        Memory forensics. vol -f memdump.raw windows.info

== CRYPTOGRAPHY ==
openssl            Crypto swiss army knife. openssl enc, openssl rsa, openssl s_client
RsaCtfTool         Automated RSA attacks. RsaCtfTool -n N -e E --attack all
sage               SageMath CAS. sage -c 'from sage.all import *; print(factor(n))'
                   ECM factoring: sage -c 'from sage.all import *; print(ecm.factor(n))'
                   LLL lattice: sage -c 'M=matrix(ZZ,[...]); print(M.LLL())'
cado-nfs           Number Field Sieve (large factoring/DLP). cado-nfs <N>
flatter            Fast LLL lattice reduction. flatter < input_matrix

== BUILD TOOLS / RUNTIMES ==
gcc / g++          C/C++ compiler. gcc -o exploit exploit.c
make / cmake       Build systems.
jq                 JSON processor. cat file.json | jq '.key'
nodejs / npm       JavaScript runtime + package manager. node script.js
sqlite3            SQLite CLI. sqlite3 file.db '.dump'

== SOURCE CODE REVIEW ==
semgrep            Pattern-based SAST across 30+ languages.
                   semgrep --config=auto path/                 (sensible defaults)
                   semgrep --config=p/security-audit path/     (security ruleset)
                   semgrep --config=p/python path/             (language-specific)
                   semgrep --config=p/owasp-top-ten path/
trailmark          Trail of Bits' code-graph analyzer (REQUIRES python3.12).
                   Polyglot: 20+ languages incl. Solidity/Cairo/Circom.
                   python3.12 -c '
                   import trailmark
                   g = trailmark.Graph.from_directory("./src")
                   print(g.attack_surface())              # entrypoints + trust
                   print(g.entrypoint_paths_to("system")) # taint paths
                   print(g.complexity_hotspots(15))       # high-complexity fns
                   print(g.functions_that_raise("ValueError"))
                   '
                   See https://github.com/trailofbits/trailmark for full API.

== NOTES ==
- Sandbox runs ARM64. Do NOT execute x86/x86-64 binaries locally.
  Use Ghidra tools for RE, and interact with remote services for dynamic analysis.
- Need a package not listed? Run `pip install <pkg>` or `uv pip install <pkg>`.
- Distfiles are at /challenge/distfiles/ (read-only)
- Write generated/repaired files to /challenge/workspace/ (writable)
- Services on the host are reachable via host.docker.internal (bridge networking).
- RsaCtfTool is at /opt/RsaCtfTool/
- CADO-NFS is at /opt/cado-nfs/
